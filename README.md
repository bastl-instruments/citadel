# Citadel

<img src="./images/readme/bastl-citadel-wavebard.png" alt="Bastl Citadel Wave Bard" style="width:150px"/>

An Eurorack adaptation of the Kastle 2 pocket-sized synthesizer platform. This repository contains schematics, panel design tools, and build resources.

For code examples, see the [Kastle 2 GitHub Repository](https://github.com/bastl-instruments/kastle2) - the code is compatible with both platforms.
